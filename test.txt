Core Architecture (from repo and docs):

Frontend: Next.js (App Router) + React + TypeScript + Tailwind CSS + shadcn/ui.
Multi-Agent Orchestration: LangGraph (state-machine graph for agent turns, discussion flow, conditional transitions).
Generation Pipeline: Two-stage async pipeline (1. Outline generation → 2. Scene creation with slides, quizzes, simulations, PBL).
Action Engine: 28+ action types (speech, whiteboard drawing, spotlight/laser pointer, TTS narration, etc.).
Playback Engine: Finite-state machine (idle → playing → live) with guards and side-effects.
Provider Abstraction: Uniform interface for LLMs, TTS, ASR, image/video generation (supports OpenAI, Anthropic, Gemini, DeepSeek, Ollama, etc.).
Export & Integration: PPTX/HTML export, OpenClaw skill for messaging apps (Feishu, Slack, Telegram, Discord).
State Management: Zustand stores for agent/UI state.
Advanced Features: MinerU for complex PDF parsing (tables, formulas, OCR), SSE streaming for real-time chat, async job polling for long-running generation.
