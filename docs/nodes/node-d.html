Performance Notes

  • MoE advantage: Only 3B of 35B parameters are active per token. This makes CPU inference viable despite the model size.
  • Bottleneck: DDR5 memory bandwidth, not compute. Increasing thread count beyond 8 shows diminishing returns.
  • KV cache: q4_0 quantization via TurboQuant. Reduces KV memory to enable longer context windows within 48GB RAM.
  • NPU status: Intel AI Boost (~11 TOPS INT8) cannot run models above 3B parameters. Excluded from inference strategy.
