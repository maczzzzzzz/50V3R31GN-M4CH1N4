Setup Runbook

  1. Rebuild to 24.11 generation: sudo nixos-rebuild switch -I nixos-config=/etc/nixos/configuration.nix -I nixpkgs=/nix/var/nix/profiles/per-user/root/channels/nixos
  2. Set boot default to new generation
  3. Reboot — verify nvidia-smi works
  4. Add Tailscale to NixOS config and rebuild
  5. Add static IP or DHCP reservation (MAC: verify via ip link)
  6. Build ik_llama.cpp with CUDA backend
  7. SCP model from staging (/mnt/d/llama.cpp/models/ on Node B)
  8. Start inference, benchmark, register with LiteLLM

NixOS Config

Flake config at nix/hosts/node-c/default.nix. On-disk config at /etc/nixos/configuration.nix. The on-disk config includes NVIDIA driver setup, Docker with NVIDIA container toolkit, and both maczz and nixos users.
