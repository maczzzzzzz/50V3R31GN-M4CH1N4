Purpose

Node A does not run model inference. The 4GB GTX 1050 Ti cannot load any useful LLM. Instead, Node A serves as:

  • hermes-lcm primary: Persistent state ledger and semantic memory store
  • KV-cache spillover target: Receives quantized KV tensors from Node D via RPC over Tailscale (TurboQuant q4_0)
  • Mesh coordination: Syncs state to Nodes B, C, D

NixOS Config

Flake-based config at nix/hosts/node-a/default.nix. Imports hermes-lcm module with sync targets:

services.hermes-lcm.syncNodes = [
  "100.66.173.31"   # Node B
  "100.102.109.81"  # Node C
  "100.105.166.45"  # Node D
];

Current Status

  • SSH: Reachable via Tailscale
  • hermes-lcm: Not yet deployed (awaiting Phase 3)
  • KV spillover: Not yet configured (awaiting llama.cpp RPC support validation)
