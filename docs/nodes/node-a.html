Purpose

Node A runs lightweight inference with Qwen3-0.6B for fast, low-latency responses. The 4GB GTX 1050 Ti cannot load larger models, so GPU inference is not used. Node A also serves as:

  • mesh-micro: Fastest inference route for simple queries
  • hermes-lcm sync target: Receives state sync from primary (Node B)
  • State persistence: Secondary storage node
