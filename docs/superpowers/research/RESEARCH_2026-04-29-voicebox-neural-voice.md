## ◈ 1. CONCEPTUAL ARCHETYPE: THE VOICE STUDIO
Voicebox is a local-first, private alternative to cloud voice services. It transitions the system from "Speech-to-Text" to a complete "Neural Voice Pipeline" with high-fidelity cloning and multi-engine synthesis.

## ◈ 2. CORE CAPABILITIES
*   **Neural Voice Cloning:** Instant cloning of voices from minimal samples. Supports 50+ presets and 23 languages.
*   **Built-in MCP Server:** Direct integration with agentic loops. Hermes can "call" Voicebox tools to synthesize high-fidelity responses.
*   **Global Dictation Ingress:** System-wide hotkey-driven speech ingestion that bypasses standard UI focus.
*   **LLM Refinement Layer:** Uses a local LLM (Qwen3) to clean up transcriptions and apply "Acoustic Personalities" before synthesis.

## ◈ 3. INTEGRATION: SOVEREIGN VOICE GATEWAY
*   **Acoustic Personality Shards:** Link [SOVEREIGN_OS] and [RED_DIRECTOR] to different high-fidelity voice clones in Voicebox.
*   **MCP Bridging:** Hermes on Node D can synthesize audio directly via the Voicebox MCP server, bypassing standard SSE string-relays.
*   **VibeVoice Synergy:**

    *   **Voicebox (Jamie Pine):** Use for high-fidelity "Final Output" synthesis and "Personality Synthesis."
