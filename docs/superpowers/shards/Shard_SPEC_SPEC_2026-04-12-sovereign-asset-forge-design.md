The **Sovereign Asset Forge** is a recursive manufacturing pipeline designed to replace all legacy circular "mook" tokens with high-fidelity, top-down tactical sprites. It also establishes a modular library of "Active Props" (including security cameras) to dress the environments generated by the Atlas Forge.

## 2. THE ST3GG TRANSFUSION PIPELINE

The system uses existing steganographic metadata to drive new generations.

### 2.1 Extraction (Harvest)
- **Script:** `scripts/forge/st3gg-transfusion.ts`
- **Logic:** Extracts the "Genetic Blueprint" (JSON) from legacy circular tokens using `crush forge extract`.
- **Data:** Faction, Role, Gear, and Rank.

### 2.2 Generation (Nano Banana 2)
- **Prompting:** Node B constructs aesthetic prompts based on extracted DNA + TTTA style anchors.
- **Reference:** Uses the existing `assets/` library as a style/perspective anchor.
- **Constraint:** Forced top-down perspective and transparent background.

### 2.3 Recursive Audit (Node A)
- **Integrity Check:** Kernel verifies perspective, gear-correspondence, and alpha channel health.
- **Verification Gate:** Must achieve an audit score of ≥ 0.8.
- **Failure State:** Triggers VSB `RE_ROLL` (0x03) with refined prompt directives.
