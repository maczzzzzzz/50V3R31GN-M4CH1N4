## ◈ EXECUTIVE SUMMARY
**Mission:** Unify the Flutter mobile app design with the WebGL Pretext HUD. Establish the mobile app as a "Lite Complete Extension" of the monolithic shroud, maintaining all current sync functionality while boosting visual and cognitive capabilities.

## ◈ CORE ARCHITECTURAL PRIMITIVES

### 1. Lite Pretext Engine (Flutter Paragraph API)
- **Concept:** Implementation of the "Pure Arithmetic" layout philosophy in Flutter.
- **Mechanism:** Using `dart:ui.ParagraphBuilder` to render dense agentic logs and documentation shards without widget tree overhead.
- **Benefit:** High-speed "Linguistic Folding" and zero-jank scrolling through massive technical vaults.

### 2. The Frosted Mobile Shroud
- **Aesthetic:** Frosted Gruvbox glassmorphism using GPU-accelerated `BackdropFilter`.
- **Unification:** Exact color parity with WebGL (#282828 background, #ebdbb2 text, #fb4934 accents).
- **Optimization:** Use of `RepaintBoundary` to isolate the high-intensity background shaders from the interaction layer.

### 3. Mobile Context DAG & Circular Vitals
- **Context:** A simplified 2.5D reasoning graph visualizing current agentic branches.
- **Telemetry:** "Circular Pressure Rings" in the header visualizing VRAM and token density parity with the main system.
