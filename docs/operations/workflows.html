Model Deployment

Download to Staging

All models download to Node B's Windows filesystem first:

# Download from HuggingFace
curl -L -o /mnt/d/llama.cpp/models/model-name.gguf \
  "https://huggingface.co/<org>/<repo>/resolve/main/model-name.gguf"

Transfer to Target Node

# SCP to target
scp /mnt/d/llama.cpp/models/model-name.gguf nixos@<tailscale-ip>:/home/nixos/llama.cpp/models/

# Verify
ssh nixos@<tailscale-ip> "ls -lh /home/nixos/llama.cpp/models/"

Smoke Test

ssh nixos@<tailscale-ip>
cd ~/llama.cpp
./build/bin/llama-cli -m models/model-name.gguf \
  -p "Hello, respond briefly." -n 64 --no-warmup

NixOS Rebuild

From Flake (Preferred)

cd ~/50V3R31GN-M4CH1N4-stable-mesh-alpha
sudo nixos-rebuild switch --flake .#node-d

From On-Disk Config (Fallback)

sudo nixos-rebuild switch \
  -I nixos-config=/etc/nixos/configuration.nix \
  -I nixpkgs=/nix/var/nix/profiles/per-user/root/channels/nixos
Common pitfall: sudo does not inherit NIX_PATH from the calling shell. Always use -I flags or set it inline with sudo env NIX_PATH=... nixos-rebuild switch.

Git Workflow

# Branch for changes
git checkout -b fix/node-c-nvidia stable/mesh-alpha

# Commit and push
git add -A && git commit -m "fix(node-c): add NVIDIA driver module"
git push origin fix/node-c-nvidia

# PR via gh CLI
gh pr create --base stable/mesh-alpha --title "fix(node-c): NVIDIA driver" --body "..."

Health Check

Quick mesh-wide connectivity test:

for node in 100.90.196.70 100.66.173.31 100.102.109.81 100.120.225.12; do
  echo -n "$node: "
  ssh -o ConnectTimeout=3 nixos@$node "hostname; nixos-version" 2>&1 || echo "UNREACHABLE"
done
