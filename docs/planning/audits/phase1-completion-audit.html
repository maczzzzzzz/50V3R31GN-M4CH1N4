3. Security Findings

CRITICAL: LiteLLM Version Mismatch (CVE UNPATCHED)

CHANGELOG v0.2.0-alpha states: "LiteLLM mesh router: Upgraded from 1.82.6 to 1.84.0 in Docker container to patch critical SQL injection vulnerability."

Actual container version: v1.82.6. The upgrade was either never applied or reverted on restart due to floating image tag main-latest.

Impact: The SQL injection CVE in litellm <1.83 remains UNPATCHED.

MEDIUM: Hardcoded API Keys in Source Control

Multiple files (sniffer/triage.py, mesh/mesh_proxy.py, mesh/proxy.yml, mesh/litellm-mesh.yaml) contain hardcoded API_KEY or MASTER_KEY values.
