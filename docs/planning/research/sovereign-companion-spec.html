◈ RESEARCH: SOVEREIGN COMPANION OVERLAY

Status: PROPOSED | Category: AGENTIC UX | Date: 2026-05-10

Executive Summary

This report evaluates the feasibility of a "Sovereign Companion"—a spawnable sidecar or toggleable workspace overlay character. Inspired by the Open Design "pet" component, this entity serves as a visual manifestation of the Quaternary Mesh state, reacting to VSB Pulse events and reasoning streams.

1. Reference Analysis: Open Design 'Pet'

The open-design/components/pet implementation provides a blueprint for Agentic UX:

  • Deterministic Identity: Character species and traits are generated via user-ID hashing, ensuring a persistent "soul" for each mesh.
  • Observer Pattern: The pet acts as a secondary LLM instance that "watches" the primary agent's telemetry and provides contextual critique or encouragement.
  • Stat-Driven Personality: Traits like Chaos, Wisdom, and Sarcasmo modulate the system prompt of the companion.
  • State Persistence: Stats are anchored in a local SQLite database, allowing for cross-session memory.

2. Proposed Sovereign Implementation

A. The Visual Layer (HUD Overlay)

The companion will be materialized as a toggleable SVG/Canvas sprite within the hermes-workspace dashboard. It will utilize Framer Motion for kinetic responsiveness to mesh events.

B. Logic Integration (VSB Artery)

Unlike standard UI pets, the Sovereign Companion is bound to the VSB Protocol:

  • Pulse Reaction: The sprite "pulses" or changes color frequency based on the 302-byte binary UDP heartbeat.
  • Thinking Visualization: When Node D enters the "Heavy Reasoning" phase (detected via reasoning_content), the companion enters an "Ascended" state.

3. Strategic Implementation Path

  1. Component Scaffolding: Create dashboard/src/components/CompanionOverlay.tsx.
  2. State Mapping: Bind the component to the useMeshStore (Zustand) to react to real-time Node A-D vitals.
  3. Persona Injection: Define a minimal "Shadow Mode" system prompt that allows the companion to comment on terminal output without interfering with mission logic.
