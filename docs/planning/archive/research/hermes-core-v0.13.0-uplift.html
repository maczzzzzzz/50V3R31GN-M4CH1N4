Hermes Core Uplift

Version 0.13.0: The Tenacity Release Audit

1. Release Summary

Released on May 7, 2026, version 0.13.0 ("Tenacity") represents a massive modernization of the Hermes core, introducing durable multi-agent coordination, persistence rewrites, and a pluggable provider architecture.

2. Key Feature Audit

  • Multi-Agent Kanban: A durable board for task delegation with heartbeats, zombie detection, and hallucination recovery.
  • Ralph Loop (/goal): Native cross-turn goal persistence, ensuring the agent remains locked on target.
  • Multimodal Perception: Integration of video_analyze for native video understanding.
  • Pluggable Providers: Decoupled inference backends (OpenRouter, Anthropic, Gemini, etc.) through the ProviderProfile ABC.
  • Checkpoints v2: Total rewrite of state persistence with real pruning and disk guardrails.

3. Sovereign Mesh Alignment

Our Quaternary Mesh (Beta v3) is now synchronized with this release. The following native surfaces are utilized:

  • Memory: Pluggable hermes-lcm (and upcoming hermes-gac) providers.
  • Routing: Native sovereign-vsb provider for mesh-wide model dispatch.
  • Security: transform_llm_output hooks for real-time intent vetting.

4. Security Hardening

The Tenacity release closed 8 P0 security vulnerabilities, including guild-scoped Discord role allowlists and TOCTOU window closures in authentication logic. Redaction is now active by default.
