Sovereign Intelligence Artery

Design Specification: High-Fidelity Research & Memory Consolidation

1. Executive Summary

The Sovereign Intelligence Artery is a distributed cognitive architecture designed to bridge Local Deep Research (LDR) with advanced Geometry-Aware Consolidation (GAC). It solves the "Compression Cliff" problem—where agentic memory loses detail over time—by decoupling logic-heavy consolidation math from encrypted state persistence.

2. Mesh Topology & Roles

The architecture leverages the physical strengths of the Quaternary Mesh:

Node Role Hardware Constraint Functional Stack
Node D Orchestrator 48GB DDR5 (NPU) LangGraph LDR Engine, GAC Dimensional Router, LLM Reasoning Core
Node A The Vault 16GB DDR4 SQLite-VSM (Encrypted), RAM-Resident Vector Index, SearXNG Proxy

3. Component Architecture

3.1 LDR Ingestion (Node D)

A multi-agent research service using the sovereign-sniffer tool. It executes asynchronous loops to search, crawl, and synthesize raw data. Output is streamed directly to the GAC Router instead of flooding the context window.

3.2 GAC Router (Node D)

Middleware that calculates the Effective Dimensionality ($d_{\text{eff}}$) of semantic clusters.

  • Centroid Action: For redundant clusters ($d_{\text{eff}} < \theta$), it emits a lossy summary.
  • Medoid Action: For diverse clusters ($d_{\text{eff}} \geq \theta$), it selects the most relevant original document as a lossless representative.

3.3 VSM Vault (Node A)

An ultra-lean service managing an encrypted SQLite instance and memory-mapped vector buffers. With 16GB RAM, the entire vector space is pinned in memory for <10ms retrieval.

3.4 Hermes Artery Plugin (hermes-gac)

A native Tenacity MemoryProvider that replaces standard RAG. It handles the Tailscale-encrypted transport and manages context injection via the prefetch() hook.

4. Data Flow (The Cognitive Loop)

  1. Ingest: LDR tools stream raw research findings into the 128k context window.
  2. Analyze: When context reaches 80%, the GAC Router analyzes the history's geometry.
  3. Consolidate: Representatives (Medoids/Centroids) are pushed to Node A via the Artery.
  4. Recall: Before each turn, hermes-gac performs a hybrid (BM25 + Vector) search on Node A and primes the next reasoning step with high-fidelity context.

5. Security & Sovereignty

  • Zero-Cloud: All inference, embedding, and storage occur on physical hardware.
  • Encrypted Transport: All mesh communication is locked within the Tailscale Artery.
  • Data at Rest: Metadata is protected via SQLCipher; vectors are stored in isolated binary buffers.
