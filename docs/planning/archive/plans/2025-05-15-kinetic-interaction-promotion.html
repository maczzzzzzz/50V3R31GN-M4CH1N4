Kinetic Interaction Promotion Implementation Plan

For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.
Goal: Promote standard UI interactions to high-fidelity Kinetic interactions using the Pretext engine in the Hermes Workspace. Architecture: Replace the standard "New Session" link in the sidebar with a KineticThoughtStream component that responds to hover. Add a background KineticThoughtStream in the WorkspaceShell to render architectural "whispers" at low opacity. Tech Stack: React, Pretext (Canvas-based rendering), Tailwind CSS.
