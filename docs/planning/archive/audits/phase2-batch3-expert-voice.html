Deployment

MATLAB MCP Bridge

  1. Add to Nix flake: crates/modules/matlab-mcp-bridge/
  2. Build binary: nix build .#matlab-mcp-bridge
  3. Register as Hermes tool in config

Goose Execution Layer

  1. Add to Nix flake: crates/modules/goose-execution/
  2. Build binary: nix build .#goose-execution
  3. Deploy on Node B as execution backend

Graphify AST

  1. Add to Nix flake: crates/modules/graphify-ast/
