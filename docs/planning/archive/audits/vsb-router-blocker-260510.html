2. BLOCKER ANALYSIS

Fault: The recent VSB refactor introduced a hard requirement for SOVEREIGN_MESH_SECRET. This variable is absent from ~/.hermes/.env.

Impact: Error messages reflect a direct connection attempt to Node D, bypassing the "Mesh" intelligence.
