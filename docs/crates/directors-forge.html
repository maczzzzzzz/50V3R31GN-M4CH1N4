Overview

Director's Forge is the tool factory for the Sovereign Mesh. It ingests GOAT toolset definitions, compiles them into executable wrapper scripts, and deploys them to mesh nodes via rsync. The forge also supports automatic API discovery using Sovereign Sniffer — observe a URL, extract API endpoints, and generate a ready-to-deploy tool definition.

Designated as the Node B tool factory, it operates from the AMD GPU inference node and distributes compiled tools across the mesh.

Key Types

ToolDefinition

Core data structure describing a tool:

  • name: String — Tool identifier (validated against path traversal)
  • description: String — Human-readable description
  • command: String — Executable command
  • args: Vec<String> — Command arguments

DirectorsForge

The forge engine. Manages library and output paths, provides the full tool lifecycle:

  • new() — Initialize from env vars (GOAT_LIBRARY_PATH, FORGE_OUTPUT_PATH)
  • ingest_tool(tool_def) — Validate, write tool.json + wrapper.sh
  • compile_tools() — Verify all wrappers are executable
  • deploy_to_mesh(nodes) — rsync to mesh nodes
  • discover_api_from_url(url) — Auto-discover via Sovereign Sniffer

Security

  • Path traversal validation — Tool names are checked for /, \, .., and null bytes
  • Shell escaping — Commands and arguments are single-quote escaped before writing wrapper scripts
  • Permission enforcement — Wrapper scripts are set to 0o755
