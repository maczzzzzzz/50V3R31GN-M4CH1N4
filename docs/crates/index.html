Validated Crates

Four active Rust crates survived the Operation Phoenix purge. Each has a Nix package definition in crates/modules/ and is exposed through the flake overlay.

CratePurposeTarget NodePhase
directors-forge Tool discovery and automatic API wrapper generation. Node B EUTHANIZED
mirage-vfs Virtual filesystem abstraction layer. Provides 4 VFS tools for transparent file access across local and remote storage. Node D Phase 3
pretext-core WASM-based browser demo framework. Renders interactive UI components from declarative specifications. Node B Phase 4
vibevoice-asr Whisper-based automatic speech recognition with emotion scoring. Part of the voice perception pipeline. Node C Phase 4
zeroboot-isolation KVM/Firecracker microVM isolation for sandboxed execution. Provides hardware-enforced boundaries for untrusted code. Node D Phase 2

Build

All crates are built via the Nix flake overlay:

# Build individual crate
nix build .#directors-forge
nix build .#mirage-vfs

# Or enter dev shell with all dependencies
nix develop

Deleted Crates

These were purged during Operation Phoenix (2026-05-12). Source code is available in git history if needed:

  • matlab-mcp-bridge — no MATLAB license on mesh
