# User Guide: Linguistic Sovereignty & Procedural Dialects (v1.9.0)

## 1. Overview
Linguistic Sovereignty is a core feature of the Neural Hive that gives every faction and district in Night City its own unique, procedurally generated dialect (conlang). This guide explains how to manage Skillstones, understand conlang speech, and utilize secret coordination channels.

## 2. Skillstones: The Language Foundation
A **Skillstone** is a compact Markdown specification that defines a conlang's phonology, grammar, and lexicon. 

- **Generation:** Skillstones are generated deterministically from a numeric seed.
- **Assignment:** Every faction in `Akashik.db` is assigned a `conlang_seed`.
- **Injection:** When an NPC from that faction speaks, their assigned Skillstone is injected into the LLM context, allowing them to speak the language fluently.

### Commands
- `/conlang <seed>`: Preview the Skillstone generated by a specific seed.
- `/conlang <seed> <payload>`: Preview the Skillstone and test linguistic steganography encoding.

## 3. Understanding NPCs
By default, NPCs may speak in their native conlang during faction-only meetings or when trying to hide information. 

### Decryption Minigame
If a player has the **Cryptography** or **Linguistics** skill, a "Decrypt" button will appear in the Foundry chat.
- **Success:** Translates the conlang text to English (or the player's primary language).
- **Critical Success:** Reveals hidden steganographic payloads (see below).

## 4. Linguistic Steganography (Secret Channels)
The Neural Hive uses **Glossopetrae** patterns to hide coordination data directly inside conlang text.

### The 9 Channels
1. **Synonym Selection:** (e.g., choosing "Zul" vs "Vex" for "Enemy").
2. **Word Order:** Permuting SVO/SOV structures.
3. **Register Toggles:** Formal vs. Informal markers.
4. **Unicode Homoglyphs:** Visually identical characters.
5. **Zero-Width Markers:** Invisible data bits.
... (and more)

### Detection
Players cannot "see" these payloads without advanced Netrunning or Cryptography checks. If detected, these payloads reveal tactical intents like "Flank the Solo" or "Requesting Backup at coordinates [X,Y]".

## 5. Implementation Notes
- **Node A (Rust):** Handles the high-performance linguistic permutation and stego encoding.
- **Node B (Node.js):** Manages the Skillstone Registry and LLM orchestration.
- **Resident Memory:** Large conlang dictionaries and the **Open-Reasoner-Zero-1.5B** model are permanently resident on Node A, ensuring zero-latency linguistic processing.
