Hardware Constraints

Node B — AMD RX 9060 XT 16GB

Vulkan backend only. ROCm is unreliable on consumer AMD cards. The 16GB VRAM fits models up to ~14B at Q6_K. Vulkan supports KHR_coopmat for matrix operations. Flash Attention is functional.

Node C — RTX 2060 6GB

Turing architecture, CUDA support. The 6GB VRAM hard-limits to 9B-class models at Q4. Anything larger requires offloading to system RAM, negating GPU acceleration. The 9B function-calling model at Q4_K_M (~5.5GB) is the practical ceiling.

Node D — Meteor Lake CPU

48GB DDR5 provides ample RAM but inference is bandwidth-bound. The MoE architecture helps: only 3B of 35B parameters are active per token, reducing per-step compute. Generation at 6.1 tok/s is slow but usable for batch reasoning tasks. The Intel NPU (11 TOPS) is excluded — insufficient for models above 3B.
