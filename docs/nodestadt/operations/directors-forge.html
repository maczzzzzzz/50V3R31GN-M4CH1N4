◈ THE DIRECTOR'S FORGE

Category: OPERATIONS | Status: MATERIALIZED

The Tool Factory

Director's Forge is an automated tool materialization engine located on Node B. It allows the Strategist to reverse-engineer external APIs and generate bit-identical Python tool wrappers for the Hermes agent.

1. Core Components

  • CLI Printing Press: Materialized in crates/modules/directors-forge/ (Rust).
  • Sniffer Integration: Utilizes Sovereign Sniffer (Stagehand) to discover API surfaces.
  • Blueprint Registry: Persistent storage of forged tool schemas in /var/lib/hermes-tools/.

2. Operational Usage

Managed via scripts/directors-forge.sh.

# Forge a new tool from OpenAPI spec
./scripts/directors-forge.sh openapi https://api.example.com/spec.json
