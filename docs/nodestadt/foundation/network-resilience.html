The Resilience Forge

The Sovereign Mesh operates over a zero-trust Tailscale Artery. To ensure stability across heterogenous environments (WSL2, Host Linux, Android), specific network "vaccinations" are applied at the Nix layer.

1. Nix Layer Optimizations

Materialized in nix/modules/network-optimization.nix.

  • TCP MSS Clamping: Automatically adjusts the Maximum Segment Size to prevent packet fragmentation over the Tailscale VPN.
  • MTU Management: Ensures the virtual tailscale0 interface remains synchronized with physical hardware MTU (1500).
  • IP Forwarding: Hardened sysctl rules for mesh-wide routing.

2. WSL2 Stabilization

Addresses the "Stuck Artery" issue where WSL2 network interfaces fail to reclaim state after host sleep/wake cycles.
