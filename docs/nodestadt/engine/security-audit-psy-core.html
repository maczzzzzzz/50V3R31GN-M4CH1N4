◈ PSY-CORE SECURITY AUDIT

Category: ENGINE | Status: MATERIALIZED

Cryptographic Audit Hook

Psy-Core is a native Hermes transform_llm_output hook that performs real-time security auditing and cryptographic verification of all agent-generated intents before they are executed on the mesh.

1. Audit Mechanisms

Materialized in sidecars/hermes-agent-nous/plugins/general/psy-core/.

  • Shadow Logic Detection: Scans agent responses for undocumented side-channel commands.
  • Secret Guard: Prevents the leakage of Tailscale auth keys or mesh-wide secrets in LLM outputs.
  • Strict Mode: When enabled in config.yaml, any unverified intent triggers an immediate session abort.

2. Engine Integration

# config.yaml Snippet
hooks:
  psy-core:
    type: general
    config:
      strict_mode: true
      audit_log_path: ~/.hermes/psy-audit.log
