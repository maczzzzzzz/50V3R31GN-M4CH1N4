Category: ENGINE | Status: MATERIALIZED

The Coordination Layer

Consensus Alignment is the distributed coordination layer that ensures all four nodes in the mesh agree on the current mission state, agent identity, and memory priority.

1. Virtualized Shared Memory

Materialized via crates/modules/consensus-alignment/ and integrated with Mirage VFS.

  • State Ledger: A Redis-backed global state that persists agent intent across node reboots.
  • Conflict Resolution: Implements a "Last Strategist Wins" protocol to resolve concurrent task assignments on Node B.

2. Physical Consensus

Utilizes Hermes-LCM (Lossless Context Management) to synchronize semantic memory blocks over the Tailscale Artery.
