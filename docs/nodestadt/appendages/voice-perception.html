1. Cognitive Voice Layer

The voice pipeline is a distributed service on Node C that wraps the Hermes core with high-fidelity amplification stack (VibeVoice + VoxCPM2).

Capture Layer

  • Omi Mobile: Flutter App → Local Backend (Node B) → Whisper (Node D).
  • HUD Ingress: Mic Ingress via Workspace → VibeVoice (Node C).

Amplification Stack (Node C)

  • ASR: VibeVoice for diarization and noise-robust transcription.
  • TTS: VoxCPM2 for 48kHz, expressive vocal return.

2. MATLAB Engineering Toolkit

Provisioned as a native MCP (Model Context Protocol) bridge on Node D, providing high-fidelity engineering and simulation capabilities.

  • Capabilities: evaluate_matlab_code, check_matlab_code, run_matlab_test_file.
  • Distribution: Managed via Nix FHS environments on Node D.

3. Oz-Skills (Procedural Memory)

Standardized agent behaviors and conventions using the Warp oz-skills framework, defining how tasks are performed without bloating the system prompt.

  • Discovery: Registered via Hermes General Plugin context.
  • Activation: Triggered via the activate_skill tool.
  • Governance: 100% Strategist audit required for all new procedural memory blocks.
