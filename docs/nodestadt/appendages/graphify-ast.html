The Code Mapper

Graphify AST is the spatial memory engine for codebase analysis. It parses source code into Abstract Syntax Trees (AST) and projects them into a persistent graph database, allowing the agent to understand multi-file dependencies instantly.

1. Technical Architecture

  • Host: Node C (Ryzen 7 3700X).
  • Logic: crates/modules/graphify-ast/ (Rust).
  • Backend: SQLite (with session-based ephemeral WAL).

2. Core Features

  • Spatial Indexing: Maps functions, classes, and variables to 3D coordinates for visualization in Sovereign Hall.
  • Dependency Tracking: Real-time tracing of call graphs across different repositories.
