◈ BROWSER AUTOMATION HARNESS

Category: APPENDAGES | Status: INTEGRATED

The Agentic Eye

The Browser Harness provides native GUI interactions and web automation capabilities, allowing the agent to "see" and interact with web-based interfaces as a human operator would.

  • Engine: Playwright-based browser-harness.
  • Deployment: Provisioned via Nix on Node B and Node D.
  • Integration: Native Hermes tool registration.

1. Core Capabilities

Full-Screen Contextual Awareness

  • Real-time screen capture and DOM analysis.
  • Visual verification of successful agent actions.
  • Automated screenshot archival for mission audits.

Autonomous Interaction

  • Dynamic form filling and multi-step navigation.
  • JS-heavy application interaction (React, Angular).
  • API endpoint verification via UI behavior.

2. Setup & Execution

The harness is isolated within a Chromium sandbox managed by the local Nix environment.

# Activation within Hermes CLI
/enable-tool browser_navigate
/navigate https://github.com/Sovereign-Machina
/capture-screenshot

3. Privacy & Gating

  • Signed Calls: Every interaction is signed by the Strategist identity.
  • Air-Gap Options: Supports local-only browser instances for sensitive document processing.
  • Audit Trail: All browser events are logged to the mesh-wide session ledger.
