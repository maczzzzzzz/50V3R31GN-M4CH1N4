Overview

Sovereign Machina is a 4-node distributed AI mesh running on physical hardware. Each node is a dedicated machine connected via a Tailscale zero-trust network. The system runs local LLM inference across heterogeneous GPUs and CPUs, orchestrated through NixOS configurations and Docker Compose runtimes.

Design principle: No cloud dependencies for inference. All models run locally. The mesh is sovereign by default.
