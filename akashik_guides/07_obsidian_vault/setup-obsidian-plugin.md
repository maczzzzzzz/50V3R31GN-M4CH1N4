# ◈ TUTORIAL: OBSIDIAN SOVEREIGN BRIDGE
**Version:** 3.8.6
**Target:** Obsidian v1.5+

## 📂 INSTALLATION PROTOCOL
1.  **Locate Plugin Shard:** Navigate to `sidecar-obsidian-plugin/` in the Sovereign root.
2.  **Build Binary:** Execute `npm install && npm run build`.
3.  **Physical Link:** 
    - Open Obsidian -> Settings -> Community Plugins.
    - Click the folder icon next to "Installed Plugins".
    - Copy the `main.js`, `manifest.json`, and `styles.css` from the build output into `.obsidian/plugins/sovereign-bridge/`.
4.  **Ignition:** Toggle "Sovereign Mesh" to ON in the Obsidian settings.

## 🕹️ COMMAND ARTERIES
- **Send to Hall:** `Ctrl+P` -> "Send Active Note to Sovereign Hall". This pushes the current buffer into the agentic context loop.
- **Vitals Monitor:** Pulsing skull in the ribbon indicates a secure connection to Node B.
