## 1. BUILDING THE PLUGIN
The Obsidian plugin must be built from source using `esbuild`.

```bash
cd sidecar-obsidian-plugin
pnpm install
pnpm run build
```

### ◈ Materialized Artifacts
The build generates two critical files:
- `main.js`
- `manifest.json`

---

## 2. INSTALLATION (DESKTOP)
1.  Navigate to your Sovereign Obsidian Vault (typically `D:\Obsidian_Sovereign_OS`).
2.  Create the plugin directory if it does not exist:
    `.obsidian/plugins/sidecar-obsidian-plugin/`
3.  Mirror the built `main.js` and `manifest.json` files into that directory.
4.  Open Obsidian -> **Settings** -> **Community Plugins**.
5.  Find **Sovereign Mesh Artery** and toggle to **ON**.

---

## 3. INSTALLATION (MOBILE)
To install on Android, you must utilize the **Sovereign Link** (Tailscale).

1.  Connect your Android device to the workstation.
2.  Navigate to the Android internal storage:
    `/Documents/Sovereign_OS/.obsidian/plugins/sidecar-obsidian-plugin/`
3.  Mirror the built artifacts from your workstation into this folder.
4.  Launch Obsidian on Android and enable the plugin via **Settings**.

---

## 4. VERIFICATION
- A **Skull Icon** will appear in the Obsidian ribbon.
- **Pulsing:** Connection to Node B (Director) is active.
- **Static:** Artery is offline; check Tailscale and Port 3011 status.
