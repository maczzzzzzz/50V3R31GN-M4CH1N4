# ◈ TUTORIAL: LOGSEQ SOVEREIGN MESH
**Version:** 3.8.6
**Target:** Logseq v0.10+

## 📂 INSTALLATION PROTOCOL
1.  **Locate Plugin Shard:** Navigate to `sidecar-logseq-plugin/`.
2.  **Physical Link:** 
    - Open Logseq -> Settings -> Plugins.
    - Click "Load unpacked plugin".
    - Select the `sidecar-logseq-plugin/` directory.
3.  **Ignition:** Ensure the HTTP API is active (Settings -> Features -> HTTP API Server).
4.  **Token Sync:** Copy your API token and set it in your environment as `LOGSEQ_TOKEN`.

## 🕹️ COMMAND ARTERIES
- **Sovereign Query:** `/Sovereign Query` in any block. Relays the block's Datalog query to the system-wide memory engine.
- **Meeting Engraver:** Meetings initiated in the Sovereign Hall are automatically shored as journal blocks with #SOVEREIGN_HALL tags.
